Welcome to PConsole Version 2.0

After developing PConsole 1.0, I realised that it seemed to be too centered around programming in files rather than in a console. Version 2.0 is here to fix this, now resembling a terminal or developer console, such as the console in source games which this is inspired by. Version 2.0 is also made in .NET Standard 2.1 rather than .NET Core 10, so that it is fully operational within the Unity game engine. It comes in the form of a class library which only needs a simple setup.
